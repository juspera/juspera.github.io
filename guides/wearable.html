Notes


The Wearable Item Guide was last updated 3/1/20.


The guide contains (almost) all the wearable items in (almost) all the permanent shops in the game that sell items for silvers (or for free, in the case of premium jewelry), including makeup and tattoos. This is not a complete guide. I have almost certainly missed a handful of items, and I have also left in a few non-wearable items just because they were cool. Most items are missing properties such as weight, worn location, or cost. This is because the guide was populated from what's already available on the GS wiki; to collect all item properties by hand would be prohibitive in terms of time expenditure.


If you would like to help fill in the guide, or can me write a script that will auto-collect item properties from shops, please contact me at gs4page@aol.com.


If you are unfamiliar with Google Sheets or Excel, please see below for how to search the guide.


Please scroll to the bottom for more info on the columns and what they contain.



How to search the guide


The Wearable Item Guide is a Google Sheet. You may use control-F to search for words, but you may also create your own filters. Two typical ways of filtering are (1) to hand-select values, and (2) to filter by a condition, such as whether rows contain a particular word.


1. To hand-select values


First, go to the Data menu, select "Filter views" and then "Create new temporary filter view."




Next, click on one of the green arrows in the column headers.




Click on the blue link for "clear."



Then, select just the value or values you'd like to see rows for, and click "OK."


If you are done filtering, you can return to normal view by clicking the X in the top-right of the black filter view border.



2. To filter by condition


First, go to the Data menu, select "Filter views" and then "Create new temporary filter view."




Next, click on one of the green arrows in the column headers.



Select "Filter by condition."



Here you have many options. Usually, you will want to choose "Text contains." In the box below that, type in the word you'd like to search for and hit the green "OK" button at the bottom.


If you are done filtering, you can return to normal view by clicking the X in the top-right of the black filter view border.


Column info


name name of item
type general item type (e.g., container, footwear, clothing)
subtype specific item type (e.g., cloak, boots, pants)
color_group generalized color (i.e., crimson, scarlet, and vermilion will all be "red"); please note that silver is under grey, gold under yellow
full_descrip full description for items that only show as short descrip when you order; this field is also used for makeup's worn appearance
town town or settlement
shop shop name
room_name room name
rnum ;go2 rnum
cost item cost; may be inexact due to racial treatment
worn where item is worn; blank means unknown
capacity item capacity (this has sometimes been estimated)
weight item weight
backroom "yes" if item is in shop backroom
customizable in what ways the item is customizable
notes additional notes, including whether the item is imbeddable or premium transport jewelry
WIG_ID internal ID number; if table is very disarrayed, sort on this to return it to some semblance of order
